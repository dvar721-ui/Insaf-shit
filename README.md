# 🎬 Movie Library - Личная кинотека

**Автор:** Ульяна Васильевна

## 📖 Описание проекта

**Movie Library** - это графическое приложение для управления личной коллекцией фильмов. Приложение позволяет добавлять, удалять, фильтровать и сохранять информацию о фильмах. Данные хранятся в формате JSON, что обеспечивает возможность импорта/экспорта коллекции.

## ✨ Функциональные возможности

### Основные функции
- **Добавление фильмов** с указанием названия, жанра, года выпуска и рейтинга
- **Просмотр списка** всех фильмов в удобной таблице
- **Удаление фильмов** из коллекции
- **Автоматическое сохранение** данных при каждом изменении

### Фильтрация
- **По жанру** - выбор конкретного жанра из выпадающего списка
- **По году выпуска** - указание диапазона лет (от и до)

### Валидация данных
- Год выпуска:
  - Должен быть целым числом
  - Не может быть меньше 1888 года (год создания первого фильма)
  - Не может быть больше текущего года
- Рейтинг:
  - Должен быть числом (целым или дробным)
  - Диапазон: от 0 до 10

### Хранение данных
- Все данные сохраняются в файл `movies.json`
- При запуске приложение автоматически загружает сохраненную коллекцию
- Формат JSON обеспечивает совместимость с другими программами
